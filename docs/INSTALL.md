##Установка
###linux debian/ubuntu:
* sudo apt-get install mongodb
* sudo apt-get install nodejs
* git clone https://bitbucket.org/severeisland/testero.git

##Запуск
###При первом запуске:

Для запуска проекта нужно по-порядку запустить:

1. run-mongodb
2. init-db (при самом первом запуске)
3. bin/development
